NEW FEATURE: message ai to create transaction
 - there will be a floating button that will open a chat with the ai, the ai will be able to create transactions based on the user's messages.
 - simple input example: user types "McDonalds 10$" and the ai will create a transaction with the name "McDonalds", the amount "10$" and the category "Food". If no account is selected, the Main Account will be used as default. If no date is selected, the current date will be used as default.
 - advanced input example: user types "Dinner, 50$, main account, yesterday, #Universal" and the ai will create a transaction with the description "Dinner", the amount "50$", account "Main Account", category "Food", date "yesterday" and tag "Universal".
 - the ai will use the same logic as the transaction creation form to create the transaction.
 - after analysing the user message, the ai will show the transaction in a preview mode, with all the fields filled, and the user will be able to edit it before saving.
 - use a cheap AI model, like Gemini Flash or something similar.
 - the ai will be available in the mobile app and in the web app.
 - the mascot image should appear in the chat window, as if the user is talking to it.
 - there should be a expandable guide to the user explaining how the messages can be sent, with examples.
 - the guide should be available in the chat window, and the user should be able to close it and open it again.
